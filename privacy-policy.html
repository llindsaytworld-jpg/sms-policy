Privacy Policy

Last updated: March 6, 2026

This privacy policy describes how HuddyLindsay ("we," "us," or "our") collects, uses, and protects information in connection with our internal SMS task reminder service (the "Service").

What the Service Does

The Service is a private, internal task reminder system used by a single authorized user. The user sends text messages containing task reminders to a designated phone number. The system processes these messages to create calendar events and sends daily morning task briefings via SMS.

Information We Collect

The Service collects only the information necessary to function:

Phone number of the sole authorized user, text message content (task descriptions and reminder requests), calendar event data created from task messages, and CRM task data retrieved for daily briefings.

How We Use Information

Information is used exclusively to provide the task reminder service to the sole authorized user. Specifically, we use it to parse task reminders from incoming text messages, create calendar events based on those reminders, compile and send daily morning task briefings, and deliver confirmation messages when tasks are added.

Information Sharing

We do not sell, rent, trade, or share any personal information with third parties for marketing or advertising purposes. Message data is processed through third-party services (Twilio for SMS delivery, Google Calendar for event creation, Anthropic for message parsing, and HubSpot for CRM task retrieval) solely to provide the Service functionality.

Data Retention

Text message content is processed in real time and is not stored long-term beyond what is necessary for service delivery and standard platform logging.

Data Security

We use industry-standard security measures including encrypted communications (HTTPS/TLS) and secure API authentication to protect data in transit and at rest.

Opt-Out

The sole user may stop receiving messages at any time by texting STOP to the Service phone number. Text HELP for assistance.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.

Contact

For questions about this privacy policy or the Service, contact us at huddylindsay@gmail.com.
